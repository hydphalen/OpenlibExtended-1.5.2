## ✨ 功能特性

- **跨平台支持** — Android、iOS、Windows、Linux、macOS 全平台覆盖
- **智能下载** — 多镜像自动切换，确保下载稳定可靠
- **高级搜索** — 按语言、格式、年份筛选，支持实时搜索建议
- **内置阅读器** — 支持 EPUB 和 PDF，自定义手势操作
- **书库管理** — 追踪藏书、发现热门书籍、导出文件
